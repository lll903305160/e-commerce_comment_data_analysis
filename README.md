# e-commerce_comment_data_analysis

中国面部彩妆市场曾经历快速发展阶段，但客户消费疲态已显，叠加消费主力人群增长放缓，预期市场将在22年后逐渐放缓。
本份报告通过商品评价文本信息来分析客户需求痛点，并给出优化建议。
所获取数据源包括某品牌及2家竞品的美妆产品评价信息，来源于天猫及京东2个平台，原始数据量约1.5w。
